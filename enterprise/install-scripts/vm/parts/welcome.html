Welcome to OpenShift

App delivery just got a whole lot easier.

This virtual machine is preconfigured as an all-in-one OpenShift Enterprise system, complete with the Eclipse IDE. From here you can develop new applications directly against the world-class OpenShift Enterprise platform-as-a-Service.

The Web Console

Click below to view the OpenShift Enterprise web console. You will need these login credentials:

  • Username: demo
  • Password: openshift
Open the Web Console

The rhc Utility

Open a terminal on this system and run the following command to get started with rhc, the command-line utility for developing apps with OpenShift Enterprise:

rhc setup --server localhost -l demo -p openshift

Securing this VM

This host is pre-configured with a host login and OpenShift user for ease of use. If it will be available on any network, the default passwords should be changed as follows.

You can change the password for the host "openshift" login from the terminal window with the passwd command.

$ passwd
(enter the current password, "openshift")
(enter your new password twice)

You can change the password for the OpenShift "demo" user (or add other logins) from the terminal window with the htpasswd command.

$ sudo htpasswd /etc/openshift/htpasswd demo
(enter your new password twice)
